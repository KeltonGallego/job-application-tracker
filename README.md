# 💼 Job Application Tracker

A modern, responsive web application for organizing and tracking your job search.

## 🌐 Live Demo

https://keltongallego.github.io/job-application-tracker/

## ✨ Features

- Add, edit and delete job applications
- Search by position or company
- Filter by status, category and priority
- Sort by date, company and priority
- List View and Kanban View
- Analytics dashboard with interactive Chart.js charts
- Success rate and interview rate metrics
- Dark Mode with saved preference
- Export and import application data as JSON
- Toast notifications for user actions
- Smooth GSAP animations
- Responsive design for desktop, tablet and mobile
- LocalStorage persistence

## 🛠 Tech Stack

- JavaScript (ES Modules)
- HTML5
- CSS3
- Vite
- Chart.js
- GSAP
- Toastify JS

## 🚀 Run Locally

```bash
npm install
npm run dev
```

## 📦 Build for Production

```bash
npm run build
```
